## 📝 Descripción del Proyecto
Este proyecto consiste en el desarrollo de un **Tablero de Control Profesional** interactivo utilizando **Streamlit**. El objetivo principal es realizar un análisis profundo del dataset *AI & Data Science Job Market (2020–2026)*, explorando tendencias salariales, roles emergentes y habilidades técnicas más demandadas en la industria actual.

El proyecto incluye:
- **Landing Page:** Una página de inicio con la presentación del proyecto.
- **Dashboard Interactivo:** Filtros dinámicos y visualizaciones avanzadas con Plotly.
- **Integración con Kaggle:** Descarga automatizada del dataset mediante `kagglehub`.
